# Organização da Entrega

## Estrutura de Pastas

Conforme o enunciado, você precisa criar uma pasta com a concatenação dos nomes dos componentes do grupo.

Exemplo: se os nomes são "João", "Maria", "Pedro", a pasta seria: `Joao_Maria_Pedro`

Dentro desta pasta:
- `Paxos/` - códigos e RESPOSTAS.pdf
- `Raft/` - códigos e resultados

## Como Organizar

1. Renomeie a pasta `ENTREGA_GRUPO` com os nomes dos membros do grupo
2. Os arquivos já estão organizados dentro das pastas corretas
3. Verifique se o `RESPOSTAS.pdf` está em `Paxos/`
